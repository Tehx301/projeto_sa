# Descrição do Projeto
Desenvolvimento de um **jogo de adivinhar a paçavra** para web.
o obijetivo é aplicar conceitos fundamentais de lógica de programação, manipulação de Dom e organização de código em um projeto real de desenvolvimento
O jogador deve descobrir uma palavra secreta tentando adivinhar letras, com um número limitado de tentativas.

## Como ira funcionar
O jogo tera uma palavra que o jogador tera que adivinhar podendo errar no maximo 7 veses, um sistema de pontuação que almentara cade ves qoe o player acerta
a letra ou a palavra alem de mostrar as lettras que ja foran e eram erradas
